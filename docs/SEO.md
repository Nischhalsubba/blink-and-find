# SEO Plan

Blink & Find now has a stronger search foundation. Search ranking is never guaranteed, because Google is not a vending machine where you insert meta tags and receive glory. But the basics are now in place.

## Implemented

- Stronger root metadata title and description.
- Expanded keyword coverage for memory, reaction, focus, and number-hunting searches.
- Open Graph and Twitter metadata for better sharing.
- JSON-LD structured data for WebSite, VideoGame, WebApplication, and HowTo.
- `/rules` page with human-readable game instructions.
- Sitemap route at `/sitemap.xml`.
- Robots route at `/robots.txt`.
- Rules page prioritized in sitemap.
- PWA manifest description improved.
- Friendly product microcopy across setup, ready, gameplay, summary, and results screens.

## Target Search Themes

Primary:

- free online memory game
- number hunting game
- find the number game
- browser memory game
- reaction time game

Secondary:

- brain training game
- visual memory game
- focus training game
- online multiplayer number game
- play memory game with friends

## Recommended Google Search Console Steps

1. Add the Cloudflare URL as a property.
2. Submit `/sitemap.xml`.
3. Inspect the home page URL.
4. Request indexing for `/` and `/rules`.
5. Check Coverage and Page Experience after Google crawls the site.
6. Watch search queries and adjust page copy based on real impressions.

## Content Growth Ideas

To rank better over time, add content pages that answer real search intent:

- `/rules` already covers how to play.
- Add `/tips` later for memory and focus strategies.
- Add `/modes` later for Same Challenge vs Live Race.
- Add FAQ sections if players ask repeated questions.

## Important Reality Check

SEO improvements help discoverability, but ranking also depends on crawlability, page performance, backlinks, user engagement, domain history, and whether Google decides to have a normal day. The app now has a good technical foundation; ranking will need time, indexing, and useful content.
