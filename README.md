# Blink & Find

Flash it. Find it. Beat the clock.

Blink & Find is a fast-paced number hunting game where players race against time to locate flashed numbers hidden inside a randomized grid.

## Stack

- Next.js
- React
- TypeScript
- Bootstrap 5
- Bootstrap Icons
- Vercel

## Planned v1 Features

- Dynamic board size
- Dynamic player count
- Dynamic round count
- Turn-based multiplayer
- Single-player practice
- Timer-based scoring
- Wrong-tap penalty
- Responsive web UI

## Development

```bash
npm install
npm run dev
```

Open the local URL shown in your terminal.
